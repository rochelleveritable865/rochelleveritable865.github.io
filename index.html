⚡ q36 - Run Large Language Models With Speed

Download q36 Engine

q36 brings high-speed text processing to your personal computer. This engine helps you run the Qwen 3.6 model on your Windows machine. It uses the power of your graphics card to provide fast answers. You do not need to install complex software libraries to get this working.

📋 System Requirements

Your computer needs specific parts to handle this software. Please confirm your system meets these needs before you start:

  • Operating System: Windows 10 or Windows 11 (64-bit).
  • Graphics Card: NVIDIA RTX 5090 or hardware that supports the Blackwell architecture.
  • RAM: 32 GB of system memory is recommended.
  • Storage: At least 50 GB of free space on a Solid State Drive.
  • Drivers: The latest NVIDIA graphics drivers must be installed.

🚀 How to Download and Install

Follow these steps to set up the software on your machine:

  1. Go to the official download page to find the installer.
  2. Locate the latest release link on that page. It will appear as a file ending in .exe.
  3. Click the link to save the installer to your Downloads folder.
  4. Double-click the downloaded file to start the installation.
  5. Follow the prompts on your screen. Choose a folder where you want to keep the program.
  6. Wait for the progress bar to finish. The engine will extract the necessary model files.

🖥️ Using the Engine

Once you finish the installation, you can launch the program from your desktop shortcut. A window will open showing the status of your connection to the model.

The first time you run the engine, it performs a check of your hardware. This ensures your graphics card interacts correctly with the software. After this check, you will see a text box. Type your request into this box and press the Enter key. The engine will process your request and display the output in the main window.

🛠️ Troubleshooting Common Issues

If you encounter errors, check these common items:

  • Graphics Errors: Ensure your NVIDIA drivers are current. You can update these through the GeForce Experience application or the NVIDIA website.
  • Slow Speeds: Close other applications that use your graphics card, such as video games or professional video editing tools.
  • Installation Fails: Ensure you have sufficient disk space on your drive. Move the installation to a drive with more capacity if necessary.
  • Permission Errors: Run the installer as an administrator by right-clicking the file and selecting "Run as administrator."
